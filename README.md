# GITSUB 

* Name of the Project
  Team No Name: GitSub

* Overview of the project
  To create a similar app to GitHub

* Link to your wireframes/prototype
  https://www.figma.com/file/8vIX1yEXd2UxdWARbWqAje/GitSub?node-id=5%3A21

* Link to the deployed project
  https://team-no-name.netlify.app/

* Link to your project board
  https://github.com/nss-evening-cohort-14/gitsub-team-no-name/projects

* Description of the user
  Anyone who wants to use the forms to create repos, projects, and packages.

* List of features
  -Navbar to jump between pages
  -Pin/unpin repos
  -Add new pinned repo
  -Select a technology when adding repo
  -Form to create and describe new repo
  -Form to create and describe new project
  -Searchbar to filter through projects
  -Form to create and describe new packages
  -Seachbar to filter through packages
  -Ability to delete package
